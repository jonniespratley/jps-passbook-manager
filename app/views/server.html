API Server

This is the resource object that contains all of the REST api methods for a full CRUD on a mongo account document. 

    // * REST METHODS:

// * POST     add             http://localhost:4040/passbookmanager/passes

// * DELETE   destroy         http://localhost:4040/passbookmanager/passes/:id
Web Service API POST Handles registering a device to recieve push notifications. DELETE Handles unregistering a device. GET Handles getting the serial numbers for Passes associated with a device. GET Handles getting the serial numbers for Passes associated with a device.
